Install Brain CLI

Get up and running in 30 seconds

Intel Macs

curl -L https://github.com/anthropics/brain-cli/releases/latest/download/brain-mac -o brain && chmod +x brain && sudo mv brain /usr/local/bin/

Apple Silicon Macs

curl -L https://github.com/anthropics/brain-cli/releases/latest/download/brain-mac-arm64 -o brain && chmod +x brain && sudo mv brain /usr/local/bin/
curl -L https://github.com/anthropics/brain-cli/releases/latest/download/brain-linux -o brain && chmod +x brain && sudo mv brain /usr/local/bin/

Download the Windows executable and add it to your PATH:

Download brain.exe

Then move it to a directory in your PATH or add the current directory to your PATH.

Build from source (requires Deno):

git clone https://github.com/anthropics/brain-cli.git && cd brain-cli && deno task build && sudo cp bin/brain /usr/local/bin/

Verify Installation

brain --version && brain --help

Quick Start

Get context management in 3 simple steps

1

Navigate to a git repository

cd your-project
2

Set up your OpenAI API key (optional)

brain config set openai-key sk-your-key-here
3

Start capturing context

brain save "debugging user authentication flow"

When you return later:

brain resume

Shows your thoughts + AI analysis + suggested next steps
