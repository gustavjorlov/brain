Quick Start

Get context management in 3 simple steps

1

Navigate to a git repository

cd your-project
2

Set up your OpenAI API key (optional)

brain config set openai-key sk-your-key-here
3

Start capturing context

brain save "debugging user authentication flow"

When you return later:

brain resume

Shows your thoughts + AI analysis + suggested next steps

Install Brain CLI

Get up and running in 30 seconds

Intel Macs

curl -L https://github.com/gustavjorlov/brain/releases/latest/download/brain-mac -o brain && chmod +x brain && sudo mv brain /usr/local/bin/

Apple Silicon Macs

curl -L https://github.com/gustavjorlov/brain/releases/latest/download/brain-mac-arm64 -o brain && chmod +x brain && sudo mv brain /usr/local/bin/
